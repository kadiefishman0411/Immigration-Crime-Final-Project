# Immigration-Crime-Final-Project
Dataset for Comm-260 Final Project examining the correlation between immigration and crime rates in the U.S.
# Overview
My project is about the correlation between crime and immigration. Due to the current political climate, there has been a consistent rhetoric claiming that immigrants have increased the risk of crime. Based on my data, I have discovered there is a very low to no correlation between the two variables.
# Data Sources
My immigration data over time comes from Migration Policy Institute, 1850-2024. The crime data comes from the FBI Crime Data Explorer.
# Methodology
I imported each data into their own tab on an excel file. The crime data was done monthly, so I found the average violent crime rate each year by combining the total from each subsection and then the average from all the months in one year. 
# Limitations
This is correlation, not causation. It says that there is a insignificant positive relationship/correlation between the two variables, but the data does not answer whether immigrants cause crime.
The numbers from both datasets are drastically different and do not show both patterns on one graph. Thus, it is necessary for the data to be standardized so it can be placed onto one graph and resemble accurate relationships and patterns next to eachother. This is a work in progress and will be updated to resemble more meaningful findings.
